# SeatCast (3인 전용 화상회의)

회사 내부에서 3명이 사용하는 간단한 Zoom 스타일 화상회의 앱입니다.

## 기능

- 최대 3명 회의실 제한
- WebRTC 기반 실시간 음성/영상
- 참석/퇴장 알림 및 참석자 목록
- 카메라 동작을 따라가는 버추얼 캐릭터 좌석 렌더링
- 마이크/카메라/버추얼 모드 토글

## 실행

```bash
npm start
```

브라우저에서 `http://localhost:3000` 접속 후 회의 코드와 이름을 입력해 참가합니다.

## 참고

- 버추얼 캐릭터 포즈 추정은 브라우저에서 MediaPipe Pose 스크립트를 로드해 처리합니다.
- WebRTC STUN 서버로 `stun:stun.l.google.com:19302`를 사용합니다.
- 사내망/방화벽 환경에서 P2P가 막히면 연결이 제한될 수 있습니다.
